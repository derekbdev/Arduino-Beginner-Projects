# Arduino-Beginner-Projects
A collection of different simple code and projects to learn Arduinou
## Starter Projects Created
* ** 3 BLINKING LEDS: ** Cycles through three different LEDS in a sequence.
* ** SOS SIGNAL LEDS : ** Blinks the built in code into Morse Code.
* ** 15 SECOND BLINKING LED: ** A Simple timer to practice delay of a function and digitalwrite functions.
## HARDWARE NEEDED
* Arduino uno (or compatible board)
* LED and Resistors
* Bread Board and Jumpwires

## HOW TO USE
1. Open the '.ino' file on arduino in the ide and create new sketch.
2. Connect your board via usb to your pc.
3. Click the **UPLOAD** button.
